## 关于我 About Me

你好,我是 Lucas,一位专注于区块链安全/密码学应用以及开发工具的技术人员.这是一个 Hugo && 所用主题为 FixIt;博客地址是: https://www.lucas6.xyz

在过去的几年中,我在以下方向持续探索：

- **区块链密码学**
- **硬件 × 冷钱包方案**
- **区块链合约**
- **助记词 × 地址派生机制**
- **区块链安全**
- **区块链的共识机制**
- **矿机 × 矿池 × 挖矿**
- **ZKP/零知识证明技术/协议**

我的目标是：**打造一个可验证/安全/高效的密码学生态工具栈**,并帮助开发者理解加密背后的逻辑,而不是仅仅使用工具.
